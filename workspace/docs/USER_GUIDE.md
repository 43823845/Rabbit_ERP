# 用户操作手册 — 功能与页面流程

## 1. 系统启动与登录

### 1.1 首次启动
```
┌──────────── 登录页 ────────────┐
│                                 │
│  ┌─ 品牌区 ─┐  ┌─ 登录区 ──┐  │
│  │ 财信ERP   │  │ 用户名    │  │
│  │ 专业外账   │  │ 密码     │  │
│  │ 管理系统   │  │ 选择账套  │  │
│  │           │  │           │  │
│  │ · 凭证管理│  │ [登录系统] │  │
│  │ · 账簿报表│  │           │  │
│  │ · 财务报表│  │ + 新建账套 │  │
│  └───────────┘  └───────────┘  │
└─────────────────────────────────┘
```

### 1.2 新建账套
1. 点击「+ 新建账套」或账套下拉框为空时自动提示
2. 填写公司信息：
   - **公司名称**（必填）
   - 联系人、联系电话
   - 地址、税号
3. 点击「创建账套」
4. 系统自动创建对应账套并选中

### 1.3 登录
- 默认账号：`admin` / `admin`
- 选择已有账套后点击「登录系统」
- 无账套时自动引导新建

### 3.3 操作流程
```
创建凭证 → 保存草稿 → 审核 → 过账
    │                     │
    └── 可编辑/删除        └── 不可编辑
```

### 3.4 分录输入规则
- 最少 2 行（借贷各至少 1 行）
- 借方合计 = 贷方合计
- 科目通过下拉选择（按树形顺序展示，搜索支持编码和名称）
- **科目显示格式**：一级科目直接显示名称；二级科目显示「父科目-子科目」
- **新增科目**：下拉列表底部「+ 新增科目」可快速创建二级科目
- 摘要可手动输入或从模板选择

## 4. 会计科目管理 (AccountSubject)

### 4.1 页面布局
```
┌────────── 会计科目 ────────────┐
│ [全部] [资产] [负债] [权益] [成本] [损益] │
│                                 │
│ 🔍 搜索...  [展开所有] [隐藏禁用] │
│                                 │
│ [+新增] [导入] [导出] [删除]       │
│                                 │
│ ┌─────────────────────────────┐ │
│ │☑│操作  │编码  │名称    │类别│…│
│ │ │+✎✕ │1001 │库存现金│资产│ │ │
│ │ │+✎✕ │1002 │银行存款│资产│ │ │  ← 有子科目的行整行浅色高亮
│ │ │     │     │  ▸支付宝  │    │ │ │  ← 展开后显示二级
│ │ │+✎✕ │1003 │其他货币│资产│ │ │
│ └─────────────────────────────┘ │
│ 显示 79 条 / 共 81 条            │
└─────────────────────────────────┘
```

### 4.2 核心功能
- **类别标签切换**：全部 / 资产 / 负债 / 权益 / 成本 / 损益
- **树形表格**：
  - **默认全部折叠**，有下级科目的行整行浅色（暖黄）背景高亮提示
  - **点击名称列任意位置**展开/折叠下级科目（非仅图标，整列可点击）
  - 展开箭头 ▸ 位于名称列左侧，展开后旋转 90° 并变为金色
- **列顺序**：选择框 → 操作(新增下级/编辑/删除) → 编码 → 名称 → 类别 → 方向 → 状态
- **内置科目**：79 个金蝶标准一级科目，不可删除（仅可新增下级）
- **用户科目**：自由创建二级科目，系统自动生成编码
- **行内操作**：一级内置科目录「+ 新增下级」；用户科目录编辑、删除
- **批量操作**：勾选后批量启用/禁用/删除
- **搜索过滤**：编码/名称搜索，展开所有级次，隐藏禁用科目

### 4.3 新增二级科目
1. 点击一级科目操作列的「+ 新增下级」按钮
2. 在弹出的表单中填写二级科目名称
3. 系统自动按 `一级编码 + 两位序号` 生成编码

### 4.4 凭证中快速新增科目
在新增凭证弹窗中，点击会计科目下拉列表底部的「+ 新增科目」：
1. 选择要扩展的一级科目
2. 填写二级科目名称
3. 确认后立即返回凭证，新科目自动出现在下拉列表中

## 6. 账簿报表 (LedgerView)

### 6.1 页面 Tab 切换
```
┌──────────── 账簿报表 ────────────┐
│ [科目余额表] [试算平衡表] [利润表] [资产负债表] │
│                                   │
│ 期间选择: [2026-06 ▾]             │
│                                   │
│ ┌───────────────────────────────┐ │
│ │         报表内容              │ │
│ └───────────────────────────────┘ │
└───────────────────────────────────┘
```

### 6.2 四大报表功能

| 报表 | 说明 |
|------|------|
| **科目余额表** | 指定期间各科目的期初、借方发生、贷方发生、余额 |
| **试算平衡表** | 验证「借方合计 = 贷方合计」，含期初/发生/期末 |
| **利润表** | 收入 - 费用 = 利润，自动按 income/expense 科目分类 |
| **资产负债表** | 资产 = 负债 + 权益，含本期利润自动并入 |

### 6.3 数据来源
```
所有报表仅统计 status = 'posted' 的凭证
期初余额来自 gl_opening_balance
发生额来自 gl_voucher_entry (通过 gl_voucher 过滤)
```

## 7. 完整业务操作流程

```
  ① 新建账套（首次）
       ↓
  ② 登录系统
       ↓
  ③ 进入工作台
       ↓
  ④ 管理会计科目（可选）
       │  └── 新增/修改/禁用科目
       ↓
  ⑤ 录入期初余额（新公司初始化）
       ↓
  ⑥ 日常凭证操作
       ├── 新建凭证（录入分录）
       ├── 修改草稿凭证
       ├── 审核凭证
       ├── 过账凭证
       └── 删除草稿凭证
       ↓
  ⑦ 月末处理
       ├── 查看试算平衡表
       ├── 调整凭证
       ├── 生成利润表 / 资产负债表
       └── 结账（关闭当前期间）
       ↓
  ⑧ 下月继续（切换期间）
```
