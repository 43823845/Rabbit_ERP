# 用户操作手册 — 功能与页面流程

## 1. 系统启动与登录

### 1.1 首次启动
```
┌──────────── 登录页 ────────────┐
│                                 │
│  ┌─ 品牌区 ─┐  ┌─ 登录区 ──┐  │
│  │ 财信ERP   │  │ 用户名    │  │
│  │ 专业外账   │  │ 密码     │  │
│  │ 管理系统   │  │ 选择账套  │  │
│  │           │  │           │  │
│  │ · 凭证管理│  │ [登录系统] │  │
│  │ · 账簿报表│  │           │  │
│  │ · 财务报表│  │ + 新建账套 │  │
│  └───────────┘  └───────────┘  │
└─────────────────────────────────┘
```

### 1.2 新建账套
1. 点击「+ 新建账套」或账套下拉框为空时自动提示
2. 填写公司信息：
   - **公司名称**（必填）
   - **联系人**（必填）
   - **企业法人**（必填）
   - 联系电话、地址
   - **税号**（必填，18位统一社会信用代码）
3. 点击「创建账套」
4. 系统自动创建对应账套并选中

### 1.3 登录
- 默认账号：`admin` / `admin`
- 选择已有账套后点击「登录系统」
- 无账套时自动引导新建

### 3.3 操作流程
```
创建凭证 → 保存草稿 → 审核 → 过账
    │                     │
    └── 可编辑/删除        └── 不可编辑
```

### 3.4 分录输入规则
- 最少 2 行（借贷各至少 1 行）
- 借方合计 = 贷方合计
- 科目通过下拉选择（按树形顺序展示，搜索支持编码和名称）
- **科目显示格式**：一级科目直接显示名称；二级科目显示「父科目-子科目」
- **新增科目**：下拉列表底部「+ 新增科目」可快速创建二级科目
- 摘要可手动输入或从模板选择

## 4. 会计科目管理 (AccountSubject)

### 4.1 页面布局
```
┌────────── 会计科目 ────────────┐
│ [全部] [资产] [负债] [权益] [成本] [损益] │
│                                 │
│ 🔍 搜索...  [展开所有] [隐藏禁用] │
│                                 │
│ [+新增] [导入] [导出] [删除]       │
│                                 │
│ ┌─────────────────────────────┐ │
│ │☑│操作  │编码  │名称    │类别│…│
│ │ │+✎✕ │1001 │库存现金│资产│ │ │
│ │ │+✎✕ │1002 │银行存款│资产│ │ │  ← 有子科目的行整行浅色高亮
│ │ │     │     │  ▸支付宝  │    │ │ │  ← 展开后显示二级
│ │ │+✎✕ │1003 │其他货币│资产│ │ │
│ └─────────────────────────────┘ │
│ 显示 79 条 / 共 81 条            │
└─────────────────────────────────┘
```

### 4.2 核心功能
- **类别标签切换**：全部 / 资产 / 负债 / 权益 / 成本 / 损益
- **树形表格**：
  - **默认全部折叠**，有下级科目的行整行浅色（暖黄）背景高亮提示
  - **点击名称列任意位置**展开/折叠下级科目（非仅图标，整列可点击）
  - 展开箭头 ▸ 位于名称列左侧，展开后旋转 90° 并变为金色
- **列顺序**：选择框 → 操作(新增下级/编辑/删除) → 编码 → 名称 → 类别 → 方向 → 状态
- **内置科目**：79 个金蝶标准一级科目，不可删除（仅可新增下级）
- **用户科目**：自由创建二级科目，系统自动生成编码
- **行内操作**：一级内置科目录「+ 新增下级」；用户科目录编辑、删除
- **批量操作**：勾选后批量启用/禁用/删除
- **搜索过滤**：编码/名称搜索，展开所有级次，隐藏禁用科目

### 4.3 新增二级科目
1. 点击一级科目操作列的「+ 新增下级」按钮
2. 在弹出的表单中填写二级科目名称
3. 系统自动按 `一级编码 + 两位序号` 生成编码

### 4.4 凭证中快速新增科目
在新增凭证弹窗中，点击会计科目下拉列表底部的「+ 新增科目」：
1. 选择要扩展的一级科目
2. 填写二级科目名称
3. 确认后立即返回凭证，新科目自动出现在下拉列表中

### 6.2 账簿功能一览

| 账簿 | 说明 | 状态 |
|------|------|:----:|
| **科目余额表** | 指定期间各科目的期初、借方发生、贷方发生、余额 | ✅ |
| **明细账** | 逐笔分录 + 余额滚动，含期初/合计/累计行 | ✅ |
| **总账** | 按一级科目汇总，含期初/本期/累计 | ✅ |
| **试算平衡表** | 验证「借方合计 = 贷方合计」（期初+发生+期末三层），含平衡状态标记 | ✅ v1.2 |
| **核算项目余额表** | 按辅助核算类别（客户/供应商等）汇总额 | ✅ |
| **核算项目明细账** | 按辅助核算维度展示逐笔明细 | ✅ |
| **核算项目组合表** | 辅助核算类别 × 科目二维交叉矩阵，含行列合计 | ✅ v1.2 |
| **利润表** | 收入 - 费用 = 利润，多级公式模板 | ✅ |
| **资产负债表** | 资产 = 负债 + 权益，利润自动并入 | ✅ |

### 6.3 试算平衡表 (v1.2 新增)

- **位置**：账簿报表 → 主账簿 → 试算平衡表
- **功能**：三层平衡验证（期初余额、本期发生额、期末余额）
- **平衡判定**：使用浮点容差 `Math.abs(借方-贷方) < 0.005` 判定是否平衡
- **状态显示**：绿色「平衡」徽章或红色「不平衡」徽章
- **导出**：支持 CSV 格式导出

### 6.4 核算项目组合表 (v1.2 新增)

- **位置**：账簿报表 → 辅助账簿 → 核算项目组合表
- **功能**：选择辅助核算类别（客户/供应商/部门/项目/职员），生成该类别下各项目值 × 各科目的二维交叉表
- **表格结构**：行 = 科目，列 = 辅助核算项目值，单元格 = 借方/贷方金额
- **合计**：每行借方/贷方合计 + 每列借方/贷方合计

### 6.5 数据来源
```
所有报表仅统计 status = 'posted' 的凭证
期初余额来自 gl_opening_balance
发生额来自 gl_voucher_entry (通过 gl_voucher 过滤)
```

### 7.2 报表功能说明

| 报表 | 说明 | 导出 |
|------|------|:----:|
| **利润表** | 按收入/费用/成本分类，多级公式计算净利润 | Excel |
| **资产负债表** | 资产 = 负债 + 权益，期末利润自动并入 | Excel |
| **所有者权益变动表** (v1.2) | 实收资本/资本公积/盈余公积/未分配利润/本年利润 期初→期末变动 | Excel |

### 7.3 通用操作

1. 点击左侧报表名称切换报表
2. 选择会计期间后自动查询
3. 点击「导出 Excel」按钮导出当前报表数据
4. 导出的 Excel 包含标准财务报表格式（表头、数据区、签名行）

## 8. 完整业务操作流程

```
  ① 新建账套（首次）
       ↓
  ② 登录系统
       ↓
  ③ 进入工作台
       ↓
  ④ 管理会计科目（可选）
       │  └── 新增/修改/禁用科目
       ↓
  ⑤ 录入期初余额（新公司初始化）
       ↓
  ⑥ 日常凭证操作
       ├── 新建凭证（录入分录）
       ├── 修改草稿凭证
       ├── 审核凭证
       ├── 过账凭证
       └── 删除草稿凭证
       ↓
  ⑦ 月末处理
       ├── 查看试算平衡表
       ├── 调整凭证
       ├── 生成利润表 / 资产负债表
       └── 结账（关闭当前期间）
       ↓
  ⑧ 下月继续（切换期间）
```
